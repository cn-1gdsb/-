## 🧪 制作前准备（必读）

- 所有器材（除废液桶和天平外）均已用**哇哈哈纯净水**清洗并消毒
- 桌面及手套已严格消毒
- **口罩一定要全程佩戴**
- **千万不要直接用手触摸任何试剂**（包括雌二醇、卡波姆等）
- 雌二醇添加量**可自定义**，但**不要过多**，推荐 **220mg**（其他物质请严格按教程用量）

## 📋 详细制作步骤

### 1. 称取卡波姆
取 **0.5g 卡波姆980**

![卡波姆](https://github.com/cn_1gdsb/雌二醇凝胶的制作/raw/main/images/IMG_7175.jpg)

### 2. 溶解卡波姆
取 **40mL 纯净水** 溶解卡波姆，搅拌至充分溶解（可搅拌一会儿或静置自然水解）。

![溶解](https://github.com/cn_1gdsb/雌二醇凝胶的制作/raw/main/images/IMG_7177.jpg)

### 3. 调雌二醇溶液
取 **220mg 雌二醇**（推荐量），加入 **63mL 95% 酒精** 搅拌溶解。

![酒精溶解](https://github.com/cn_1gdsb/雌二醇凝胶的制作/raw/main/images/IMG_7181.jpg)

### 4. 加入其他成分
- 加入 **1.4g 氮酮**
- 加入 **1.55g 甘油（1,2-丙二醇）**

![添加成分](https://github.com/cn_1gdsb/雌二醇凝胶的制作/raw/main/images/IMG_7180.jpg)
![添加甘油](https://github.com/cn_1gdsb/雌二醇凝胶的制作/raw/main/images/IMG_7183.jpg)

### 5. 加入三乙醇胺
加入 **0.7g 三乙醇胺** 调节 pH 值。

![三乙醇胺](https://github.com/cn_1gdsb/雌二醇凝胶的制作/raw/main/images/IMG_7186.jpg)

### 6. 测量 pH 值
使用 pH 试纸测量溶液 pH 值。

![测量 pH](https://github.com/cn_1gdsb/雌二醇凝胶的制作/raw/main/images/IMG_7187.jpg)
![pH 试纸](https://github.com/cn_1gdsb/雌二醇凝胶的制作/raw/main/images/IMG_7188.jpg)

### 7. 混合与装瓶
充分混合均匀后装瓶，**建议冰箱冷藏保存**。

![混合](https://github.com/cn_1gdsb/雌二醇凝胶的制作/raw/main/images/IMG_7184.jpg)
![成品](https://github.com/cn_1gdsb/雌二醇凝胶的制作/raw/main/images/IMG_7190.jpg)

## 🧼 制作后清理

- **桌面清理**：直接用**酒精 + 纸巾** 擦干净即可
- **器材清洗**：建议先用水 + 酒精浸泡一段时间，再进行彻底清洗
- 废液请按危险废弃物要求妥善处理
